### 公式大法

**[count] + operator + [count] + motion/text-object**

- d3w # 删除 3 个单词
- dfi / d2fi # f 表示 find
- di” / d2i” # i 表示 inside, a 表示 around, 注意你不一定要先移动到”里面进行操作

### 移动

- hjkl/fFtT
- ctrl+o/i
- m’
- gd/gr

### 编辑

- cmp + snippet
- macros
- :normal :s
